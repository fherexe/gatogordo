🐱 CSS Parallax Experiment

Projeto simples criado para testar efeitos visuais em CSS, principalmente comportamento de backgrounds fixos e efeito parallax durante o scroll da página.

A página utiliza diferentes seções com imagens e GIFs de gatos para demonstrar como o CSS pode criar uma experiência visual dinâmica sem necessidade de JavaScript.


🛠️ Tecnologias utilizadas

HTML5

CSS3

Google Fonts

Fonte utilizada:

Google Fonts – Roboto Flex

Principais conceitos aplicados:

background-attachment: fixed

background-position

background-size

nth-of-type

media queries para responsividade


🎯 Objetivo do projeto

Este projeto foi criado como experimento de aprendizado em CSS, explorando:

efeitos visuais com scroll

manipulação de background


📚 Aprendizados

comportamento de backgrounds no CSS

criação de efeito parallax sem JavaScript

uso de GIFs e imagens como elementos visuais

estruturação de páginas com múltiplas seções
